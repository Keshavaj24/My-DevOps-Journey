1. / - Main directory (Root folder)
2. /home - user files like doc, projects,
3. /etc - Configuration files ex: network settings, user settings, service configs
4. /bin - basis cmd like ls, cp, mv
5. /sbin - system cmd for sys admin (reboot, shutdown) used by admin root
6. /var - changing data /var/log (This is where logs are stored)
8. /tmp - Temporary file stored and may delete them automatically
9. /usr - Installed programs /usr/bin
10. /boot - Files needed to start linux contains Kernel, bootloader files
11. /dev - Hardware devices as files  /dev/sda (hard disk)
12. /mnt and /media - used for external storage (USB, Hard drives, ISO files)
13. /root - admin HOME
