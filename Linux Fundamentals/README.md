### What is Linux?
Linux is an open-source operating system software that manages the Hardware and run programs.
Linux Kernel - It is a core part of OS that manages:

- CPU, MEMORY, DISK, NETWORK DEVICES and other Pheripheral devices etc.

### What is Servers?
A server is a machine that runs 24/7 responds to requests over a network.
It must be:

- Stable, Secure, Fast, Easy to automate, Cheap to operate

### Why Linux is matters? or Why should i use Linux?

90% of Servers works on Linux system because 

- Linux system can run for months and years w/o reboot, If your server randomly reboot - your company loses revenue.
- Linux has strong permission model it stricts file permissions and user separation etc.
- Linux is an open-source and reduces infrastructure cost massively.
- Servers must be automated Linux supports
    - Shell scripting, Cron jobs, System services, CLI based control
    - Try managing 500 servers manually it's difficult and time consuming
    - So automation is survival.
