## Learning Structure 
### Prerequisites
- Firstly, Understand what is DevOps? Why do we need DevOps?
- Understand the flow SDLC (Software Development Lifecycle)
  
### Basic Foundation
- Linux Fundamentals
- Git & Version Control (using Github)
- Bash Scripting
- Networking Fundamentals - IP address, Subnets, ports, OSI model, DNS etc

### Core DevOps Tools
- Docker & Containers
- Kubernetes
- AWS Cloud services
- Infrastructure as Code / CI-CD (any one)
  - Terraform
  - Jenkins
  - Github Actions
- Python
  
### Monitoring & Observability
- Prometheus
- Grafana

## Other Areas to focus on

## End Goal
Become a skilled DevOps Enginner with strong knowledge of Linux, cloud, containers, automation, and monitoring.
